// features

What it does.

Two surfaces, one Rust core. No Electron. No Node.

Terminal pager

Pipe any Markdown file through md for rich ANSI output. Integrates with your existing pager (less, bat). Ratatui-powered interactive view with scroll and search.

ratatui ANSI

Desktop preview

Native window with live-rendered preview, document outline, frontmatter metadata panel, file watching, and drag-and-drop file open. No browser required.

Tauri 2 WebView

Shared Rust core

Both surfaces use the same parser and renderer contracts. Tested with fixtures. Adding a new rendering target means implementing one trait, not duplicating logic.

Rust tested

File watching

Open a file and edits propagate instantly. Works with any editor. The desktop app auto-reloads on save; the terminal view polls for changes in the background.

notify hot reload

YAML frontmatter

Parses and renders YAML front matter as a structured metadata sidebar. Useful for static-site authors and note-taking workflows using tools like Obsidian or Jekyll.

YAML metadata

Mermaid diagrams

Fenced mermaid blocks render as SVG in the desktop preview. The terminal surface outputs ASCII-art approximations compatible with any monospace environment.

SVG ASCII fallback
// architecture

Small crates, clear boundaries.

The workspace is split into focused crates. Each layer depends only on the layer below it.

markdown-core parser · renderer traits · test fixtures
terminal renderer ratatui · ANSI · ASCII diagrams
HTML renderer WebView · syntax highlight · Mermaid
md CLI unified binary
Tauri app macOS · Linux
cargo workspace

All crates live in a single Cargo workspace. cargo build --workspace builds everything in one shot.

renderer trait

The Renderer trait in markdown-core is the only contract the leaf crates implement — swapping rendering targets requires no changes to the parser.

test fixtures

Shared fixture files guarantee rendering parity. A failing fixture in one surface is a bug in that surface alone, not the parser.

// quick start

Build and run.

Requires Rust stable and Cargo. Tauri build requires the platform WebView runtime.

01

Quality checks

cargo fmt --all -- --check

cargo clippy \
  --workspace \
  --all-targets \
  --all-features \
  -- -D warnings

cargo test \
  --workspace \
  --all-features
02

Build artifacts

# CLI binary
cargo build --release -p markdown-cli

# Desktop app + DMG
cd crates/markdown-app
cargo tauri build
cargo tauri build \
  --bundles dmg
03

Usage

# render to terminal
md README.md

# pipe to pager
md README.md | less -R

# open desktop app
markwell path/to/file.md

# watch mode
md --watch README.md
