Your docs stack,orchestrated.

MD Star is a document-native toolchain for engineers. It combines desktop editing, terminal reading, and one Rust parser core so markdown behavior stays consistent across interfaces.

Features

A compact, high-signal surface for writing and reviewing technical docs at speed.

Terminal Reader

md file.md with consistent table handling and pager-friendly output.

Desktop Edit + Preview

Autosave, outline navigation, file association open flows, and rapid iteration loops.

Shared Parser Core

One semantic Rust pipeline powers both terminal and desktop behavior.

Mermaid Support

Diagram blocks render across both interfaces without custom user transforms.

Workspace Navigation

Move through linked docs quickly with document-centric traversal patterns.

Format Expansion

Roadmap includes YAML, JSON, and XML readers with the same ergonomics.

Install

Clone, build, and run in terminal or desktop mode.

$ git clone https://github.com/danieldear/markwell
$ cd markwell
$ cargo build --release -p mdstar-app
$ ./target/release/md README.md
$ cargo run -p mdstar-app -- --app
