Capabilities

Built for serious
local Markdown work.

01

Desktop preview

Open Markdown files in a native window with rendered preview, outline navigation, metadata, file watching, and save/reload controls.

02

Terminal workflow

Use md from the shell for formatted output, pager integration, ratatui viewing, split editing, and ASCII Mermaid diagrams.

03

Shared Rust core

Desktop and terminal surfaces use the same parser and renderer contracts, backed by fixtures and integration tests.
