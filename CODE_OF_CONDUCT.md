# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and focus on technical outcomes.
- Give actionable feedback.
- Accept feedback with professionalism.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Personal attacks or intimidation.
- Publishing private information without permission.
- Any conduct that creates an unsafe environment.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that violate this Code of Conduct.

## Reporting

Report issues to project maintainers through private channels when available. Include context and links so maintainers can investigate quickly.
